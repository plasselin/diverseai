Fully functional chatbot using the OpenAI API.

Choose between GPT-4 and GPT-4 mini, use text-to-speech to hear the model talk, and speech-to-text to have a real-time conversation with Chat GPT. You can save up to 5 conversation threads linked to your user profile and come back later to pick up the session where you left it. Use documents or pictures to fully utilize OpenAI's toolset (requires the GPT-4 model) and use the temperature slider to customize the response accuracy/creativity levels.

This website uses a customized version of the Poseidon template from PrimeNG.
