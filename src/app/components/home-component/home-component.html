Developer Portfolio

This website was first created as a side project to learn the Django framework as well as the OpenAI API. Along the way, I figured I would need a portfolio website, so I decided to use this project as its base and will add new features along the way.

If you are here, that means I am currently unemployed, so I have to limit the API calls for each visitor. I do not set a hard limit on tokens since the chatbot uses TTS and STT, and that would make the demo hard to evaluate, so I gave each visitor up to 40 prompts they can use to check how I implemented the OpenAI API with Angular 17, a Django backend, and a PostgreSQL database to handle the models.

I have decided to leave the Store as is so that you may see the State with Redux and evaluate my implementation of NGRX. You should also be able to see the JSON objects in the console when using the chatbot.
